Thẻ span là 1 thẻ để chứa văn bản được sử dụng gom nhóm các chữ vào 1 nhóm để tiện cho việc style
    
        Khái niệm thuộc tính trong css
        - Thuộc tính trong html nằm trong thẻ mở của thẻ html và được sử dụng để bổ sung tính chất đặc điểm cho thẻ html
        ví dụ thuộc tính lang="en" cho thẻ html 
        thuộc tính href="" cho thẻ a 
        thuộc tính src="" cho thẻ img 
        - Thuộc tính trong css
        CSS có chức năng trang trí định dạng cho thẻ html về màu sắc , kích thước , cách hiển thị , bố cục ... 
        Thuộc tính trong css được sử dụng để chỉ ra 1 tính chất trong css 
        ví dụ thuộc tính màu sắc trong css là color và giá trị là red , green , yellow ... 

        - chèn css vào html thông qua thuộc tính html có tên là style
        background-color thuộc tính màu nền trong css
